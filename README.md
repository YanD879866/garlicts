### Garlicts框架简介

轻量级分布式web框架，使用简洁的注解，无任何xml配置，不依赖spring，无学习成本。 

框架具备ioc、aop、mvc、分布式、插件化、安全检测等能力。

初衷：一个优秀的软件公司得有自己的技术积累，根据不同需要，不断订制改造和优化，既提高了开发效率，又增加了盈利。

### Garlicts框架的特色

1、使用更简单。
2、提供多方的解决方案，解决问题更简单。
