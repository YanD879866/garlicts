### Garlicts框架简介

轻量级的web框架，使用简洁的注解，无任何xml配置，不依赖spring，但保留springmvc的使用习惯，最大程度的降低大家的学习成本。 

框架具备ioc、aop、mvc、分布式、插件化等能力。

初衷：软件公司得有自己的技术积累，得有自己的内部框架，根据不同需要，不断订制改造和优化，既提高了项目的开发效率，又增加了盈利。
